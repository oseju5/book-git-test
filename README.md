# 주제: 슬롯 머신

# 주제 선택 이유:
타이밍 게임은 어떻게 구현해야 할지 감이 바로 오지 않아,
로직이 바로 생각나는 슬롯 머신 구현을 시도했습니다.

# 기능 원리

1. 이모티콘 리스트를 배열에 담아 저장한다.
2. setInterval 함수로 각각의 아이템을 200ms 마다 반복해서 수정한다.
   이때 반복 실행되는 함수는 이모티콘 리스트 배열 크기 내 랜덤한 정수를 반환,
   해당 값을 인덱스로 하여 텍스트 값을 수정하는 함수이다.
3. 모든 아이템이 돌아간 후, setTimeout으로 딜레이 시킨 결과 출력 함수를 실행한다.
4. 각 아이템을 중복을 허용하지 않는 배열에 담고(Set) 해당 배열의 사이즈를 비교하여 결과를 조건부 출력한다.
   사이즈(중복되지 않는 요소의 개수)가 3개라면 꽝, 2개라면 아깝다, 1개라면 잭팟.
5. 버튼을 여러 번 눌러 게임이 반복실행되는 것을 방지하기 위해, 전역 변수로 현재 게임이 실행중인지를 감시한다.
   게임이 실행중이라면 alert로 안내창을 띄우고 함수를 종료한다.
